# bmi-calculator
Simple calulator app based on node.js

## Usage
Clone it
```bash
git clone https://github.com/dixit-kmt/bmi-calculator.git
```

### Package Installation
```bash
cd /path-to-repo/bmi-calculator #replace path
```
```bash
npm install express
```
### Execution
```bash
node calculator.js
```
Open browser and paste the following url
```url
http://127.0.0.1:3000/
```
